Programação aplicada às famílias de dispositivos: nomear certo para escolher certo.

Pseudocódigo

Escolha da família

if need_high_flow_dialysis:
    choose(HD_family)
elif long_term_intermittent and quality_of_life_matters:
    consider(port)
elif long_term_central and arm_route_is_good:
    consider(PICC)
elif acute_critical_multidrug:
    choose(non_tunneled_cvc)
elif long_term_exteriorized_is_better_than_reservoir:
    consider(tunneled_family)
else:
    revisit_need_and_context()
Desafios

Pense em espécies, não em rótulos vagos

Inputterapia, frequência, manutenção, horizonte temporal, fluxo.
Erro clássicotratar família como detalhe do sítio.
Outputfamília de dispositivo coerente com o problema.
Problemas aplicados

Três exercícios de algoritmo

Problema 1

Escreva `chooseLongTermDevice()` para onco intermitente vs antibiótico prolongado.

Problema 2

Monte a tabela `therapy_to_family` incluindo HD, vasoativo, quimio e NPT.

Problema 3

Crie um alerta `notJustAnotherCentral()` para HD, port e tunelizados.
