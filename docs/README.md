# Docs Index

Este diretorio concentra a documentacao da workspace.

## Taxonomia

- `docs/foundation/`
  identidade, tese, schema e norte de produto
- `docs/reviews/`
  diagnosticos e revisoes de estado
- `docs/planning/`
  planos, matrizes e backlog de execucao
- `docs/ops/`
  operacao local, deploy, PR e handoff
- `docs/qa/`
  relatorios gerados de walkthrough e validacao
