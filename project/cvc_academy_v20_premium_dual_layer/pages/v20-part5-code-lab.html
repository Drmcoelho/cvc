Programação aplicada à confirmação por imagem: artefatos, malposição, bundle e conduta.

Pseudocódigo

Algoritmo de confirmação

if ultrasound_view_is_poor:
    optimize_probe(gain, depth, pressure, angle)
elif needle_disappears:
    correct_plane_and_visibility()
elif wire_path_is_unclear:
    stop_and_reconfirm()
elif xray_shows_strange_trajectory:
    assess_malposition_hypothesis()
elif pleural_complication_suspected:
    escalate_to_thoracic_problem()
else:
    confirm_bundle_and_document()
Desafios

Pense em algoritmo interpretativo

InputUS, RX, trajetória, ponta, pleura, artefato.
Erro clássicoconfundir imagem bonita com confirmação suficiente.
Outputseguir, corrigir, reavaliar ou escalar.

Problema 2

Problema 3

Crie um bundle de confirmação em pseudocódigo que não termine só no RX.
